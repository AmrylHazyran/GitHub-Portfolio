04 04_projects.sh ✕
evidence_of_learning // academic + personal projects
FYP Python Forensics
ForensiX — Digital Forensics Data Recovery Tool

A multi-threaded Python/Tkinter desktop application for forensic disk imaging and automated data recovery. Parses partition tables and recovers deleted files from unallocated space using pytsk3. Features thread-safe live progress tracking, automated chain-of-custody reporting with MD5/SHA-256 integrity hashing, and pure Python decryption of VeraCrypt-encrypted drives (AES-256-XTS, PBKDF2-SHA512) — feeding decrypted data directly to pytsk3 via a custom VeraCryptImg subclass, without any external executables.

Python tkinter pytsk3 AES-256-XTS PBKDF2-SHA512 cryptography hashlib threading
