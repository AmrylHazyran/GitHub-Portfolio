# Amryl Hazyran - Professional Portfolio Website

A clean, responsive, and professionally designed personal portfolio website tailored for a student specializing in Cybersecurity and Digital Forensics. 

## 🌐 Overview

This portfolio is built to effectively showcase academic achievements, specialized technical skills (particularly in digital forensics and cryptography), practical projects, and a clear career roadmap. The design utilizes a modern, dark-themed aesthetic that remains highly readable and professional without relying on distracting visual gimmicks.

## ✨ Features

* **No Frameworks:** Built entirely from scratch using semantic HTML5, modern CSS3, and vanilla JavaScript.
* **Fully Responsive:** Adapts seamlessly to all screen sizes, from desktop monitors to mobile devices.
* **Smooth Navigation:** Features a fixed, blurred navigation bar with offset smooth scrolling to ensure section headers are never obscured.
* **Interactive Elements:** Includes CSS-based skill bar animations that trigger when scrolled into view using the `IntersectionObserver` API.
* **Evidence Integration:** Contains a dedicated section specifically designed to host project repositories, PDF case studies, and certificate verifications.

## 🚀 How to Use

Simply download or clone the repository and open the `index.html` file in any modern web browser. No compilation, package managers, or server setup is required.

To customize the content:
1. Open `index.html` in your preferred code editor.
2. Search for the placeholder links (e.g., `#` in the `Evidence` section) and replace them with your actual Google Drive/GitHub URLs.
3. Replace the placeholder image text in the `.hero-photo` container with an actual `<img src="profile.jpg">` tag when you have a professional headshot ready.

## 🎨 Typography & Colors

* **Fonts:** `Space Grotesk` (Headings), `Inter` (Body Text), `JetBrains Mono` (Technical details/labels). All served via Google Fonts.
* **Color Palette:** * Backgrounds: Deep Midnight Blues (`#0b0d14`, `#131627`)
  * Accents: Violet (`#7c3aed`), Cyan (`#22d3ee`), Emerald Green (`#34d399`)
  * Text: Soft Lavenders and Cool Greys for high contrast without eye strain.
